# meteor-tutorial-cli

CLI for creating Meteor Tutorials

### Install

```bash
npm install meteor-tutorial-cli
```

### Usage


#### Patch file

```bash
meteor-tutorial patch [filename]
```

#### Git Log to file

```bash
meteor-tutorial log [filename]
```
